## Transok 0.4.8

### 新增
- macOS / Windows 多平台自动打包流水线，产物命名统一为 `transok-<version>-<platform>-<arch>.<suffix>`。
- macOS DMG 内置「修复已损坏.command」，一键解决未签名应用的 quarantine 拦截。

### 下载
- macOS（Apple Silicon）：`transok-0.4.8-darwin-arm64.dmg`
- macOS（Intel）：`transok-0.4.8-darwin-amd64.dmg`
- Windows：`transok-0.4.8-windows-amd64.msi`

### 备注
未做代码签名 / 公证。macOS 首次启动若提示「已损坏」，请双击 DMG 内的"修复已损坏.command"。
