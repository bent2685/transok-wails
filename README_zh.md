# 关于

**Transok** 是一个局域网文件共享小工具。它不会占用多余的存储空间，对传输文件数量和大小没有限制。同时分享文件时可以设置密码，防止文件被非法下载。

---

# Features

- [x] 文件上传功能
- [x] 开启分享功能
- [x] 文件删除和清空
- [x] 文件下载页
- [x] 深色模式
- [x] 纯文字分享
- [x] i18n
- [x] 自定义端口
- [x] 文件加密分享
- [ ] 查找局域网内正在共享的设备[重做]
- [ ] 在 Finder/资源管理器中右键添加
