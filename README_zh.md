# 关于

**Transok** 是一个局域网文件共享工具，可以通过唯一的共享密码来分享文件。对文件数量和大小没有限制。

---

# Features

- [x] 文件上传功能
- [x] 开启分享功能
- [x] 文件删除和清空
- [x] 文件下载页
- [x] 深色模式
- [ ] 在 Finder/资源管理器中右键添加
- [ ] 文件加密分享
