# Transok Download Page

A modern file download web interface for [Transok](https://github.com/bent2685/transok-wails). This is a companion frontend application that provides a clean, responsive download page for shared files.

## About

This project serves as the web frontend for Transok's file sharing functionality. It's built with React and designed to be embedded within the main Transok Wails application or served as a standalone web page.

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Framer Motion
